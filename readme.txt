== Description ==
Send Slack notifications for WooCommerce events like new orders, low stock, and customer activity. Group notifications by thread and format messages using Slack blocks and markdown for clarity.

== Installation ==
1. Upload the plugin files to `/wp-content/plugins/`.
2. Activate through the 'Plugins' screen.
3. Configure under **Settings > Slack Notifier for WooCommerce**.
